# b2m

`b2m`은 `BLF` 파일을 `MAT` 형식으로 변환하기 위한 Windows 프로그램입니다.

차량 로그나 CAN 데이터 작업 과정에서 수집된 `BLF` 파일을 MATLAB 환경에서 바로 활용할 수 있도록 `MAT` 파일로 변환할 수 있습니다. 여러 개의 `BLF` 파일과 `DBC` 파일을 함께 불러와 작업할 수 있고, 변환 옵션을 통해 시간축 처리 방식과 보간 방식도 조정할 수 있습니다.

## 주요 기능

- `BLF` 파일을 `MAT` 파일로 변환
- `DBC` 파일을 함께 불러와 신호 해석 지원
- 여러 파일을 한 번에 추가하고 일괄 변환 가능
- 시간축 생성 방식, 보간 방식, 결측값 처리 방식 설정 가능
- 릴리즈 버전 확인 및 업데이트 기능 지원

## 최신 버전

- 현재 기준 최신 앱 버전: `1.1.13`
- 설치 파일 이름: `b2m_Setup.exe`

최신 설치 파일은 [Releases](../../releases) 페이지에서 다운로드할 수 있습니다.

## 설치 방법

1. [Releases](../../releases) 페이지로 이동합니다.
2. 최신 버전의 `b2m_Setup.exe` 를 다운로드합니다.
3. 설치 파일을 실행한 뒤 안내에 따라 설치합니다.
4. 설치가 끝나면 프로그램을 실행합니다.

## 릴리즈 노트

버전별 변경 사항은 GitHub Releases 본문과 앱 내부 릴리즈 노트 안내를 통해 확인할 수 있습니다.
